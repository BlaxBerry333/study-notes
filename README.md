# Study Notes

Personal notes app built with VitePress, featuring a Warhammer 40K Cogitator cyberpunk theme.

## Tech Stack

- [VitePress](https://vitepress.dev/) - Static site generator
- [Vue 3](https://vuejs.org/) - Custom theme components
- [TypeScript](https://www.typescriptlang.org/) - Type-safe configuration
- [Markdown](https://daringfireball.net/projects/markdown/) - Documentation format

## Development Scripts

| Command           | Description                |
| ----------------- | -------------------------- |
| `npm run dev`     | Start development server   |
| `npm run build`   | Build for production       |
| `npm run preview` | Preview production build   |
| `npm run check`   | Run Biome check and CSpell |
| `npm run format`  | Format code with Biome     |
