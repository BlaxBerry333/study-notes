# JavaScript

动态类型、基于原型、支持多范式的脚本语言

::: warning 特点

- **动态类型**: 变量类型在运行时确定，灵活但需注意隐式转换
- **基于原型**: 通过原型链实现继承，而非传统的类继承
- **事件驱动 + 异步**: 单线程 + Event Loop + 非阻塞 I/O
- **一等公民函数**: 函数可以赋值给变量、作为参数传递、作为返回值
:::
